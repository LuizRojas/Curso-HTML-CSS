Citações

Caso deseje citar algo, existem duas maneiras de se fazer isso:
Pode-se utilizar a tag <q>Texto aqui</q> (para citações simples).

Códigos-fonte

Caso deseje exibir códigos-fonte em sua página web, pode-se utilizar a tag <code>Código-fonte aqui</code>
nome = 'Peter' print(f'Olá {nome}!') Porém como é possível visualizar, o código não possui tabulação, além de ficar confuso para o leitor. Para contornar este problema, recomenda-se o uso aninhado de tags, combinando-as da seguinte maneira:
<pre><code>Código-fonte aqui</code></pre> 

            
nome = 'Peter'
print(f'Olá {nome}')
